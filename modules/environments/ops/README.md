# operations

Operations environment resources and configurations.

## Overview

This module contains the basic resources and configurations for the operations environment. It provides foundational infrastructure that can be referenced by scenario modules.

## Resources

- VPC and networking components
- Basic IAM roles and policies
- Environment-specific configurations

## Usage

This module is typically referenced by scenario modules that need to deploy resources in the operations environment. It provides the foundational infrastructure that scenario modules build upon.

## Requirements

- AWS provider configured for operations account
- Operations account ID

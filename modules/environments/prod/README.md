# prod

Production environment resources and configurations.

## Overview

This module contains the basic resources and configurations for the production environment. It provides foundational infrastructure that can be referenced by scenario modules.

## Resources

- VPC and networking components
- Lambda functions and related resources
- Basic IAM roles and policies
- Environment-specific configurations

## Usage

This module is typically referenced by scenario modules that need to deploy resources in the production environment. It provides the foundational infrastructure that scenario modules build upon.

## Requirements

- AWS provider configured for prod account
- Production account ID
